Spesifikasi Lingkungan :
* Sudah meng-install GNU Prolog Console
* OS   : Windows XP, 7, 8, 10 dan UNIX
* RAM  : Minimal 3 GB

Cara mengeksekusi program ini sebagai berikut.
1. Bukalah 'GNU Prolog Console' terlebih dahulu.
2. Tekanlah 'File'.
3. Tekanlah 'Consult' yang muncul.
4. Carilah file yang bernama 'main.pl' yang berada di dalam folder ini 
   dan tekanlah 'Open'.
5. Program sudah bisa jalan dan kata-kata untuk perintahnya (beberapa) sudah ditampilin ke layar.
6. Jika tersesat dalam permainannya, bisa memberi perintah 'help.' untuk mendapatkan informasi-informasi command.
7. Jika ingin keluar dari program, bisa memberi perintah 'quit.'.
